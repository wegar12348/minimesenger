# misegesait — R&DMESEGE Messenger

> Полноценный веб-мессенджер с двумя визуальными стилями

---

## Структура проекта

```
misegesait/
├── index.html          ← Главная страница (вся разметка)
├── .htaccess           ← Apache: кэш, защита, редиректы
├── css/
│   └── style.css       ← Все стили + обе темы
├── js/
│   ├── storage.js      ← Слой данных (localStorage)
│   ├── auth.js         ← Авторизация / регистрация
│   ├── chat.js         ← Логика чата, рендер, polling
│   ├── ui.js           ← Рендер UI, контакты, тема, toast
│   └── app.js          ← Точка входа, обработчики событий
└── php/
    └── api.php         ← Серверный API (PHP + JSON-файлы)
```

---

## Режимы работы

### Режим 1: localStorage (по умолчанию — без сервера)
Всё хранится в браузере. Идеально для демонстрации на одном устройстве.
- Просто откройте `index.html` в браузере
- **Чтобы видеть "других пользователей"** — откройте сайт в двух вкладках одного браузера, зарегистрируйте разных пользователей

### Режим 2: PHP API (реальный многопользовательский режим)
Разные устройства видят друг друга.

## Загрузка на Sprinthost

### Шаг 1 — Загрузка файлов

1. Войдите в панель управления Sprinthost → **Файловый менеджер**
2. Перейдите в папку `public_html` (корень вашего сайта)
3. Загрузите все файлы проекта, сохраняя структуру папок:
   ```
   public_html/
   ├── index.html
   ├── .htaccess
   ├── css/style.css
   ├── js/storage.js
   ├── js/auth.js
   ├── js/chat.js
   ├── js/ui.js
   ├── js/app.js
   └── php/api.php
   ```

> **Совет:** Заархивируйте папку в ZIP, загрузите архив и распакуйте через файловый менеджер Sprinthost.

### Шаг 2 — Создать папку data (для PHP-режима)

В файловом менеджере создайте папку: `public_html/php/data/`

Установите права доступа (chmod): **755**

Sprinthost → Файловый менеджер → папка `data` → Свойства → Права → 755

### Шаг 3 — Проверить PHP

В Sprinthost убедитесь, что включён PHP 7.4+. Это настраивается в разделе "PHP" панели управления.

### Шаг 4 — Открыть сайт

Перейдите по адресу вашего домена: `https://yourdomain.ru`

## Тестирование

### Быстрый тест (локально, без сервера)

1. Откройте `index.html` напрямую в браузере (Chrome/Firefox)
2. Зарегистрируйтесь: логин `user1`, имя `Алиса`, пароль `1234`
3. Откройте сайт в **новой вкладке**
4. Зарегистрируйтесь: логин `user2`, имя `Боб`, пароль `1234`
5. Выберите собеседника в боковой панели → пишите сообщения
6. Во второй вкладке выберите первого пользователя → сообщения появятся через ~1.5 сек

### Переключение тем

- На экране входа: кнопка **«⇄ Переключить стиль»**
- В чате: кнопка **⇄** в шапке сайдбара

## Безопасность

- Пароли в localStorage хранятся с простым хэшированием (только для демо!)
- В PHP-режиме используется `password_hash()` + HMAC-токены
- Для production смените `SECRET` в `php/api.php`
- Добавьте HTTPS (Sprinthost предоставляет бесплатный SSL через Let's Encrypt)
